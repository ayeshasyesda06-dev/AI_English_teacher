# AI English Teacher

A comprehensive English language learning system that combines four specialized teaching skills into a single, seamless learning experience.

## Overview

The AI English Teacher provides an automated, integrated approach to English language learning featuring:

- **English Level Analyzer** - Diagnostic assessment of your English proficiency level (CEFR A1-C1)
- **Vocabulary Generator** - Level-appropriate vocabulary generation (25 words per session)
- **Vocabulary Test** - Comprehensive testing with multiple question types
- **Sentence Correction** - Grammar and error correction with explanations
- **English Learning Journey** - Master skill that orchestrates all components in optimal sequence

## Quick Start

```bash
/english_learning_journey
```

This will initiate a complete 10-15 minute English learning session that:
1. Assesses your English level
2. Generates appropriate vocabulary
3. Tests your vocabulary knowledge
4. Provides sentence corrections
5. Delivers a comprehensive feedback report

## Usage

### For Individual Skills

Each skill can be used independently:

- `/english_level_analyzer` - Assess your English level
- `/vocabulary_generator [LEVEL]` - Generate vocabulary at specific level
- `/vocabulary_test` - Test vocabulary knowledge
- `/sentence_correction` - Correct your English sentences

### For Complete Learning Experience

- `/english_learning_journey` - Complete integrated learning session

## Learning Principles

The system is built on proven language learning science:

- **Diagnostic Assessment** - Understanding your current level before learning
- **Spaced Repetition** - Testing enhances memory retention
- **Retrieval Practice** - Multiple retrieval methods (MCQ, fill-blank, sentence-making)
- **Corrective Feedback** - Immediate error correction
- **Cumulative Learning** - Each phase builds on previous outcomes

## System Requirements

The system assesses and teaches at all CEFR proficiency levels:

## Recommendations

- **Frequency**: Use the English Learning Journey once per week
- **Practice**: Focus on identified weakness areas between sessions
- **Retention**: Review vocabulary from previous sessions
- **Progress**: Advance to higher levels when consistently scoring 85%+ on tests
