# START HERE - English Learning Journey Quick Start

## What You Have

A complete, automated English learning system with 5 skills:

1. **English Learning Journey** (NEW) - Master skill that runs everything
2. **English Level Analyzer** - Assesses your proficiency
3. **Vocabulary Generator** - Creates vocabulary lists
4. **Vocabulary Test** - Tests your knowledge
5. **Sentence Correction** - Corrects your English

---

## How to Use (3 Simple Steps)

### Step 1: Open Claude Code
Navigate to your project folder:
```
D:\AI_English_teacher
```

### Step 2: Type This Command
```
/english_learning_journey
```

### Step 3: Follow the Instructions
- Answer 5 English questions
- Review 25 vocabulary words
- Take a 25-question test
- Submit sentences for correction

**Total time: 10-15 minutes**

---

## What You'll Get

After completing the journey, you'll receive:

1. **Your English Level** (A1-C1 on international scale + star rating)
2. **Your Strengths** (what you do well in English)
3. **Your Weaknesses** (what to improve)
4. **25 Vocabulary Words** (matching your level)
5. **Your Test Score** (how well you know the vocabulary)
6. **Corrected Sentences** (with explanations of errors)

## Frequently Asked Questions

**Q: Do I need to prepare anything?**
A: No. Just type the command and answer honestly.

**Q: What if I don't know an answer?**
A: Say you don't know. This is honest assessment.

**Q: How long does it take?**
A: About 10-15 minutes total for all phases.

**Q: Can I use a dictionary?**
A: Not during the test. The test measures what you know now.

**Q: Can I redo it?**
A: Yes! Run it weekly to track progress.

**Q: What if I fail the test?**
A: That's okay. It shows you what to practice. Try again next week!

**Q: Can I use this on my own computer?**
A: Yes. All skills are installed locally.

## When to Use

### Weekly Routine (Recommended)
- **Once per week:** Run `/english_learning_journey` for full assessment
- **Daily:** Use individual skills for focused practice

### As You Progress
- **Week 1:** Establish your baseline level

- **Week 4:** Run again to see improvement
- **When ready:** Move to higher difficulty level

## If Something Doesn't Work

**Command not recognized:**
- Make sure you typed exactly: `/english_learning_journey`
- The `/` must be at the beginning

**Stuck on a question:**
- Take your time
- Answer naturally
- There's no time limit

**Test too easy or hard:**
- Easy? Try: `/vocabulary_generator B1` (higher level)
- Hard? Try: `/vocabulary_generator A1` (lower level)

## Optional: Automatic Startup

Want the English Learning Journey to run automatically when you open Claude Code?

1. Open your Claude settings file
2. Add this configuration:
```json
{
  "hooks": {
    "on_open": "/english_learning_journey"
  }
}
```
3. Save and restart Claude Code

Now it runs automatically every time you open your project!
