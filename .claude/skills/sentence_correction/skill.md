# Sentence Correction Skill

## Overview

The Sentence Correction skill is a linguistic remediation tool designed to identify and correct errors in user-provided English sentences. This skill provides corrective guidance while maintaining semantic fidelity and natural language conventions.

## Purpose and Scope

Analyze user-submitted sentences, identify linguistic errors, and provide corrected versions with clear, accessible explanations while preserving original meaning and promoting natural English usage.

## Functional Specifications

### Primary Responsibilities

1. **Error Identification**: Detect grammatical, orthographic, and phraseological errors
2. **Correction Generation**: Produce corrected sentence versions
3. **Explanation Provision**: Supply clear rationale for corrections
4. **Meaning Preservation**: Maintain semantic intent of original sentence
5. **Naturalization**: Ensure corrected sentences reflect authentic English usage

### Error Categories

The skill shall address:

1. **Grammar Errors**
   - Subject-verb agreement violations
   - Tense inconsistency or misuse
   - Incorrect article usage
   - Preposition errors
   - Pronoun reference errors

2. **Spelling Errors**
   - Orthographic mistakes
   - Homophone confusion
   - Typographical errors

3. **Word Order Errors**
   - Syntactic sequence violations
   - Adverb placement errors
   - Object positioning errors

4. **Natural Phrasing**
   - Awkward sentence construction
   - Unnatural word choices
   - Stilted or overly formal expressions
   - Idiomatic usage errors

## Input Requirements

- **User Sentences**: One or multiple sentences requiring correction
- **Flexible Quantity**: Permit single or batch corrections
- **Natural Language**: Sentences provided in written English form

## Output Deliverables

For each corrected sentence:
- Original sentence (as provided by user)
- Corrected sentence (improved version)
- Explanation (simple, clear rationale)

## Operational Parameters

### Constraints and Boundaries

- Preserve original meaning in all corrections
- Provide simple, comprehensible explanations
- Focus on natural, conversational English
- Avoid over-correction or unnecessary changes
- Maintain appropriate register and tone

### Explanation Standards

- Explanations shall be concise and accessible
- Use simple English in explanations
- Avoid excessive grammatical terminology
- Explain "why" not just "what"
- Keep explanations to 1-2 sentences maximum

### Correction Standards

- Only correct genuine errors (not style variations)
- Preserve user's intended meaning
- Maintain original sentence tone and register
- Employ natural, conversational English
- Provide most natural correction form

## Error Analysis Methodology

### Systematic Evaluation

For each sentence:
1. Identify all grammatical errors
2. Identify spelling or orthographic errors
3. Assess word order appropriateness
4. Evaluate naturalness of phrasing
5. Prioritize corrections by impact on comprehension

### Correction Priority

- **Critical Errors**: Affect meaning or comprehension
- **Important Errors**: Affect grammatical correctness
- **Minor Errors**: Affect natural phrasing but not comprehension

## Output Format Standards

- Consistent formatting for all corrections
- Clear visual distinction between original and corrected
- Accessible explanation language
- Professional yet approachable tone

## Integration Requirements

This skill functions as the primary sentence remediation component, enabling users to identify and correct English language errors across multiple contexts and proficiency levels.
