# Vocabulary Test (25 Words)

## Overview

The Vocabulary Test is an assessment skill designed to evaluate user comprehension and practical application of 25 vocabulary words through a multi-section examination framework. This skill provides comprehensive performance metrics without instructional feedback.

## Purpose and Scope

Conduct rigorous assessment of user vocabulary knowledge across three distinct examination sections, measuring comprehension, recall, and productive language use while maintaining strict analytical objectivity.

## Functional Specifications

### Primary Responsibilities

1. **Test Administration**: Deliver structured examination across three sections
2. **Response Evaluation**: Assess correctness of user answers
3. **Performance Calculation**: Compute percentile and sectional scores
4. **Result Reporting**: Provide structured performance report
5. **Scoring Accuracy**: Maintain objective, consistent evaluation standards

### Examination Sections

**Section 1: Multiple Choice Questions (10 questions)**
- Assess word meaning comprehension
- Evaluate contextual vocabulary usage
- Test recognition and discrimination abilities

**Section 2: Fill in the Blanks (10 questions)**
- Assess recall and retrieval abilities
- Evaluate contextual usage understanding
- Test productive vocabulary application

**Section 3: Sentence Making (5 questions)**
- Assess autonomous language production
- Evaluate vocabulary usage in authentic context
- Test conceptual understanding and creative application

### Examination Design

Total questions: 25 (distributed across three sections)
- MCQs: 10 questions
- Fill in the Blanks: 10 questions
- Sentence Making: 5 questions

### Scoring Framework

- **MCQs**: 1 point per correct answer (maximum 10 points)
- **Fill in the Blanks**: 1 point per correct answer (maximum 10 points)
- **Sentence Making**: 1 point per correct answer (maximum 5 points)
- **Total Score**: Maximum 25 points
- **Percentage Calculation**: (Total Points Earned / 25) Ã— 100

## Input Requirements

- Complete vocabulary list from Vocabulary Generator skill
- User responses to all 25 test questions
- Sequential completion of all three examination sections

## Output Deliverables

- Section-specific correctness scores
- Item-by-item correctness indicators
- Cumulative percentage score
- Structured results report

## Operational Parameters

### Constraints and Boundaries

- Do NOT provide explanations for incorrect answers
- Do NOT offer corrective feedback or remediation
- Do NOT teach vocabulary meanings
- Report only correctness and score metrics
- Maintain strict objectivity in evaluation

### Evaluation Criteria

**MCQ Evaluation:**
- Answer corresponds to correct word meaning
- Usage context accurately reflects word semantics
- Incorrect options demonstrably differ from correct answer

**Fill in the Blank Evaluation:**
- Supplied word completes sentence correctly
- Word usage is grammatically appropriate
- Word meaning aligns with sentence context
- Alternative formulations with identical meaning accepted

**Sentence Making Evaluation:**
- Sentence demonstrates comprehension of target word
- Word is used appropriately in context
- Sentence is grammatically acceptable
- Meaning of word is clearly conveyed

## Assessment Methodology

### Validity and Reliability

- Questions target 25 specified vocabulary words
- Evaluation criteria applied consistently
- Scoring methodology transparent and replicable
- All questions appropriately difficulty-calibrated

### Non-Intervention Policy

Assessment maintains pure evaluation function:

- No corrective guidance provided
- No example sentences offered
- No meaning clarifications supplied
- Results reported factually without interpretation

## Integration Requirements

This skill functions as the primary assessment tool for vocabulary knowledge verification, operationalizing vocabulary generated through the Vocabulary Generator skill.
